Les différentes façons dont les cartouches peuvent être chargées dans JS7800 sont détaillées tout au long du reste de cet onglet. JS7800 devrait être compatible avec la grande majorité des ROMs de cartouches disponibles pour l'Atari 7800 (à la fois des versions originales et homebrew).

Les ROMs de cartouches Atari 7800 ont typiquement une extension de fichier .bin ou .a78. L'extension .a78 est préférée car elle contient des informations d'en-tête qui fournissent à l'émulateur des informations sur la façon dont la cartouche particulière devrait être interprétée. Il est également utile de noter que JS7800 supporte les ROMs de cartouches qui sont contenues dans des fichiers .zip.

Menu Déroulant de Cartouches

Le menu "Déroulant de Cartouches" (surligné en rouge ci-dessous) contient une liste de cartouches Atari 7800 à sélectionner.

Sélectionner Cartouche Atari 7800...

Par défaut, la liste de cartouches est remplie avec une sélection d'excellents titres homebrew. Pour charger une cartouche, sélectionnez simplement l'élément approprié dans le menu.

Des "listes de cartouches" personnalisées sont également supportées. Pour plus d'informations sur la création et le chargement de listes de cartouches personnalisées, consultez la page GitHub des Listes de Cartouches.

Sélectionner Fichiers Locaux

Le bouton "Sélectionner Fichier Local" (surligné en rouge ci-dessous), est utilisé pour sélectionner et charger un fichier ROM de cartouche Atari 7800 qui est situé sur l'appareil local.

Sélectionner Cartouche Atari 7800...

Une fois le bouton pressé, un sélecteur de fichiers sera affiché qui permet la sélection d'un fichier local. Par défaut, le sélecteur de fichiers limite les fichiers affichés aux extensions .a78, .bin, .zip, et .json.

Le sélecteur de fichiers local peut également être utilisé pour sélectionner un fichier de "liste de cartouches" (extension de fichier .json). Sélectionner un fichier de "liste de cartouches" mettra à jour le contenu du Menu Déroulant de Cartouches. Pour plus d'informations sur les listes de cartouches, consultez la page GitHub des Listes de Cartouches.

Sélectionner Fichiers Distants

Le bouton "Sélectionner Fichiers Distants" (surligné en rouge ci-dessous), est utilisé pour spécifier l'URL d'une ROM de cartouche Atari 7800 distante.

Sélectionner Cartouche Atari 7800...

Une fois le bouton pressé, une boîte de dialogue de saisie sera affichée qui permet de spécifier l'URL vers un fichier ROM distant. Par exemple, entrer l'URL suivante causera le chargement du jeu homebrew, "bonQ" :

http://raz0red.github.io/js7800/roms/bonq.a78

Le sélecteur de fichiers distant peut également être utilisé pour sélectionner un fichier de "liste de cartouches" distant (extension de fichier .json). Sélectionner un fichier de "liste de cartouches" mettra à jour le contenu du Menu Déroulant de Cartouches. Pour plus d'informations sur les listes de cartouches, consultez la page GitHub des Listes de Cartouches.

Glisser-Déposer Fichiers

Glisser et déposer des ROMs de cartouches Atari 7800 locales peut être utilisé comme alternative à la sélection de fichiers via le navigateur de fichiers local (voir Sélectionner Fichiers Locaux).

Comme démontré ci-dessus, les fichiers ROM locaux peuvent être chargés en les glissant et déposant simplement sur l'émulateur depuis l'Explorateur de Fichiers sous Windows ou Finder sur Mac.

En plus des fichiers ROM de cartouches, les fichiers de "liste de cartouches" (extension de fichier .json) peuvent également être glissés et déposés sur l'émulateur. Glisser et déposer un fichier de "liste de cartouches" causera la mise à jour du contenu du Menu Déroulant de Cartouches. Pour plus d'informations sur les listes de cartouches, consultez la page GitHub des Listes de Cartouches.

Glisser-Déposer Liens

Glisser et déposer des liens qui pointent vers des ROMs de cartouches Atari 7800 distantes peut être utilisé comme alternative à la spécification d'URLs via le sélecteur de fichiers distant (voir Sélectionner Fichiers Distants).

Comme démontré ci-dessus, les ROMs peuvent être chargées en glissant et déposant simplement des liens de fichiers distants sur l'émulateur. Dans l'exemple spécifique ci-dessus, un lien est glissé depuis le Forum de Programmation Atari 7800 d'AtariAge, une excellente source pour les titres homebrew en développement et terminés. Il est également utile de noter que certains liens sur le forum AtariAge nécessitent que le glissement soit effectué via le lien de "nombre de téléchargements" versus le lien du titre du jeu (comme montré ci-dessus).

En plus des liens de fichiers ROM de cartouches, les liens de fichiers de "liste de cartouches" peuvent également être glissés et déposés sur l'émulateur. Glisser et déposer un lien de fichier de "liste de cartouches" causera la mise à jour du contenu du Menu Déroulant de Cartouches. Pour plus d'informations sur les listes de cartouches, consultez la page GitHub des Listes de Cartouches.

Paramètres de Requête

JS7800 supporte l'ensemble suivant de paramètres de requête. Les paramètres de requête peuvent être ajoutés à l'URL de JS7800 pour contrôler diverses options de l'émulateur. Par exemple, le paramètre de requête cart peut être utilisé pour forcer le chargement d'une cartouche spécifique lorsque JS7800 est initialement affiché.

cart
Le paramètre de requête cart peut être utilisé pour charger une cartouche spécifique lorsque la page de l'émulateur est initialement chargée. L'URL suivante causera le chargement du jeu homebrew, "bonQ" :

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78

cartlist
Le paramètre de requête cartlist peut être utilisé pour charger une liste de cartouches dans le Menu Déroulant de Cartouches lorsque JS7800 est initialement chargé. L'URL suivante causera le remplissage d'une liste de démos homebrew dans le menu déroulant de cartouches :

https://raz0red.github.io/js7800/?cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Pour plus d'informations sur les listes de cartouches, consultez la page GitHub des Listes de Cartouches.

Des paramètres de requête multiples peuvent être combinés pour définir plus d'une option lorsque JS7800 est chargé. L'URL suivante chargera un jeu spécifique, "bonQ", et une liste de cartouches contenant des démos homebrew :

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78&cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Puisque ajouter des paramètres de requête résultera typiquement en URLs longues, l'utilisation du service de raccourcissement d'URL de GitHub, git.io, est recommandée. L'URL suivante est la version raccourcie de l'exemple ci-dessus, contenant les deux paramètres de requête cart et cartlist :

https://git.io/JfcmO
