Die verschiedenen Wege, wie Kartuschen in JS7800 geladen werden können, werden im Rest dieses Tabs detailliert beschrieben. JS7800 sollte mit der überwiegenden Mehrheit der verfügbaren Atari 7800 Kartuschen-ROMs kompatibel sein (sowohl originale Einzelhandels- als auch Homebrew-Versionen).

Atari 7800 Kartuschen-ROMs haben typischerweise eine .bin oder .a78 Dateierweiterung. Die .a78 Erweiterung ist bevorzugt, da sie Header-Informationen enthält, die dem Emulator Informationen darüber geben, wie die spezifische Kartusche interpretiert werden sollte. Es ist auch erwähnenswert, dass JS7800 Kartuschen-ROMs unterstützt, die sich in .zip Dateien befinden.

Kartuschen-Dropdown-Menü

Das "Kartuschen-Dropdown"-Menü (unten in Rot hervorgehoben) enthält eine Liste von Atari 7800 Kartuschen zur Auswahl.

Atari 7800 Kartusche auswählen...

Standardmäßig wird die Kartuschenliste mit einer Auswahl exzellenter Homebrew-Titel gefüllt. Um eine Kartusche zu laden, wählen Sie einfach den entsprechenden Eintrag im Menü aus.

Benutzerdefinierte "Kartuschenlisten" werden ebenfalls unterstützt. Für weitere Informationen zum Erstellen und Laden benutzerdefinierter Kartuschenlisten, siehe die Kartuschenlisten GitHub-Seite.

Lokale Dateien auswählen

Der "Lokale Datei auswählen" Button (unten in Rot hervorgehoben), wird verwendet, um eine Atari 7800 Kartuschen-ROM-Datei auszuwählen und zu laden, die sich auf dem lokalen Gerät befindet.

Atari 7800 Kartusche auswählen...

Sobald der Button gedrückt wird, wird ein Datei-Auswähler angezeigt, der die Auswahl einer lokalen Datei ermöglicht. Standardmäßig beschränkt der Datei-Auswähler die angezeigten Dateien auf .a78, .bin, .zip, und .json Dateierweiterungen.

Der lokale Datei-Auswähler kann auch verwendet werden, um eine "Kartuschenlisten"-Datei auszuwählen (.json Dateierweiterung). Die Auswahl einer "Kartuschenlisten"-Datei wird den Inhalt des Kartuschen-Dropdown-Menüs aktualisieren. Für weitere Informationen zu Kartuschenlisten, siehe die Kartuschenlisten GitHub-Seite.
