Barra de Controles

A "barra de controles", que está localizada diretamente abaixo da tela do emulador JS7800, é usada para controlar vários aspectos da emulação, incluindo acesso aos diálogos de configurações e ajuda.

Os botões e controles que compõem a barra de controles são descritos abaixo.

Pausa a emulação do cartucho atual. Uma vez pausado, este botão pode ser usado para retomar a emulação.
Alterna a saída de som ligada e desligada.
Reinicia o cartucho atual. Isso é equivalente a realizar um recarregamento e início limpo do cartucho atual.
O botão SELECT representa o botão físico "SELECT" que está presente no console Atari 7800. O comportamento deste botão é específico do cartucho, mas é tipicamente usado para permitir a seleção das configurações atuais do jogo (níveis de dificuldade, etc.).
O botão RESET representa o botão físico "RESET" que está presente no console Atari 7800. O comportamento deste botão é específico do cartucho, mas é tipicamente usado para encerrar o jogo atual (se houver um em progresso).
Representa o interruptor de dificuldade esquerdo físico que está presente no console Atari 7800. O comportamento deste interruptor é específico do cartucho.
Representa o interruptor de dificuldade direito físico que está presente no console Atari 7800. O comportamento deste interruptor é específico do cartucho.
Abre a página Tabela de Líderes Global JS7800. Esta página fornece a capacidade de visualizar pontuações altas globais e melhores jogadores. O conjunto inicial de pontuações exibidas na tabela de líderes padrão para o jogo que está sendo jogado atualmente no JS7800 (se ele suportar pontuações altas globais). Para mais informações sobre a tabela de líderes global, consulte a aba "Pontuações Altas" à esquerda. Um dump bruto de jogos e variantes suportados (identificados por hash MD5 do cartucho sem cabeçalho) pode ser encontrado aqui.
Exibe este diálogo "Ajuda".
Exibe o diálogo "Configurações". Veja a aba de ajuda "Diálogo de Configurações" à esquerda para informações detalhadas.
Alterna entre modos de tela cheia e não tela cheia.
