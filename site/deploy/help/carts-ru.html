Различные способы загрузки картриджей в JS7800 подробно описаны в остальной части этой вкладки. JS7800 должен быть совместим с подавляющим большинством ROM картриджей, доступных для Atari 7800 (как оригинальные розничные, так и homebrew релизы).

ROM картриджей Atari 7800 обычно имеют расширение файла .bin или .a78. Расширение .a78 предпочтительно, поскольку содержит заголовочную информацию, которая предоставляет эмулятору информацию о том, как конкретный картридж должен интерпретироваться. Также стоит отметить, что JS7800 поддерживает ROM картриджей, содержащиеся в файлах .zip.

Выпадающее Меню Картриджей

Выпадающее меню "Картриджи" (выделено красным ниже) содержит список картриджей Atari 7800 для выбора.

Выбрать Картридж Atari 7800...

По умолчанию список картриджей заполняется подборкой отличных homebrew тайтлов. Чтобы загрузить картридж, просто выберите соответствующий элемент в меню.

Пользовательские "списки картриджей" также поддерживаются. Для дополнительной информации о создании и загрузке пользовательских списков картриджей, обратитесь к странице GitHub Списки Картриджей.

Выбрать Локальные Файлы

Кнопка "Выбрать Локальный Файл" (выделена красным ниже), используется для выбора и загрузки ROM файла картриджа Atari 7800, который расположен на локальном устройстве.

Выбрать Картридж Atari 7800...

После нажатия кнопки будет показан селектор файлов, который позволяет выбрать локальный файл. По умолчанию селектор файлов ограничивает отображаемые файлы расширениями .a78, .bin, .zip, и .json.

Локальный селектор файлов также может быть использован для выбора файла "списка картриджей" (расширение файла .json). Выбор файла "списка картриджей" обновит содержимое Выпадающего Меню Картриджей. Для дополнительной информации о списках картриджей, обратитесь к странице GitHub Списки Картриджей.

Выбрать Удаленные Файлы

Кнопка "Выбрать Удаленные Файлы" (выделена красным ниже), используется для указания URL удаленной ROM картриджа Atari 7800.

Выбрать Картридж Atari 7800...

После нажатия кнопки будет показано диалоговое окно ввода, которое позволяет указать URL к удаленному ROM файлу. Например, ввод следующего URL вызовет загрузку homebrew игры "bonQ":

http://raz0red.github.io/js7800/roms/bonq.a78

Удаленный селектор файлов также может быть использован для выбора удаленного файла "списка картриджей" (расширение файла .json). Выбор файла "списка картриджей" обновит содержимое Выпадающего Меню Картриджей. Для дополнительной информации о списках картриджей, обратитесь к странице GitHub Списки Картриджей.

Перетащить Файлы

Перетаскивание локальных ROM картриджей Atari 7800 может быть использовано как альтернатива выбору файлов через локальный браузер файлов (см. Выбрать Локальные Файлы).

Как показано выше, локальные ROM файлы могут быть загружены просто перетаскиванием их на эмулятор из Проводника в Windows или Finder на Mac.

В дополнение к ROM файлам картриджей, файлы "списка картриджей" (расширение файла .json) также могут быть перетащены на эмулятор. Перетаскивание файла "списка картриджей" вызовет обновление содержимого Выпадающего Меню Картриджей. Для дополнительной информации о списках картриджей, обратитесь к странице GitHub Списки Картриджей.

Перетащить Ссылки

Перетаскивание ссылок, указывающих на удаленные ROM картриджей Atari 7800, может быть использовано как альтернатива указанию URL через удаленный селектор файлов (см. Выбрать Удаленные Файлы).

Как показано выше, ROM могут быть загружены просто перетаскиванием ссылок удаленных файлов на эмулятор. В конкретном примере выше, ссылка перетаскивается из Форума Программирования Atari 7800 AtariAge, отличного источника как для разрабатываемых, так и завершенных homebrew тайтлов. Также стоит отметить, что некоторые ссылки на форуме AtariAge требуют выполнения перетаскивания через ссылку "счетчик загрузок" вместо ссылки названия игры (как показано выше).

В дополнение к ссылкам файлов ROM картриджей, ссылки файлов "списка картриджей" также могут быть перетащены на эмулятор. Перетаскивание ссылки файла "списка картриджей" вызовет обновление содержимого Выпадающего Меню Картриджей. Для дополнительной информации о списках картриджей, обратитесь к странице GitHub Списки Картриджей.

Параметры Запроса

JS7800 поддерживает следующий набор параметров запроса. Параметры запроса могут быть добавлены к URL JS7800 для контроля различных опций эмулятора. Например, параметр запроса cart может быть использован для принудительной загрузки конкретного картриджа когда JS7800 изначально отображается.

cart
Параметр запроса cart может быть использован для загрузки конкретного картриджа когда страница эмулятора изначально загружается. Следующий URL вызовет загрузку homebrew игры "bonQ":

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78

cartlist
Параметр запроса cartlist может быть использован для загрузки списка картриджей в Выпадающее Меню Картриджей когда JS7800 изначально загружается. Следующий URL вызовет заполнение списка homebrew демо в выпадающем меню картриджей:

https://raz0red.github.io/js7800/?cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Для дополнительной информации о списках картриджей, обратитесь к странице GitHub Списки Картриджей.

Множественные параметры запроса могут быть объединены для установки более одной опции когда JS7800 загружается. Следующий URL загрузит конкретную игру "bonQ" и список картриджей содержащий homebrew демо:

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78&cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Поскольку добавление параметров запроса обычно приводит к длинным URL, использование сервиса сокращения URL GitHub, git.io, рекомендуется. Следующий URL является сокращенной версией примера выше, содержащей оба параметра запроса cart и cartlist:

https://git.io/JfcmO
