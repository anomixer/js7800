Panoramica JS7800

JS7800 è una porta JavaScript migliorata dell' emulatore ProSystem Atari 7800 che è stato originariamente sviluppato da Greg Stanton e successivamente portato su più piattaforme.

Caratteristiche:

  • Tracciamento punteggi alti globali (per giochi compatibili con HSC)
  • Mappature tastiera personalizzabili
  • Compatibilità gamepad (con supporto dual analogico per giochi stile Robotron)
  • Supporto schermo intero
  • Multipli rapporti d'aspetto
  • Capacità di abilitare/disabilitare filtri video
  • Supporto drag and drop per file locali e link file remoti
  • Supporto liste cartucce
  • Supporto light gun (via mouse)
  • Supporto modulo espansione (XM)
  • Commutazione banche migliorata e rilevamento cartucce

Per funzionare correttamente e senza intoppi (nessun lag, ecc.), JS7800 richiede una versione aggiornata di un browser moderno (Chrome, Firefox, Safari) su un sistema configurato correttamente.
