Barra de Controles

La "Barra de Controles" directamente debajo de la pantalla del emulador JS7800 se usa para controlar varios aspectos de la emulación, incluyendo el acceso a diálogos de configuraciones y ayuda.

Los botones y controles que componen la barra de controles se describen a continuación.

Pausa la emulación del cartucho actual. Después de pausar, este botón puede usarse para reanudar la emulación.
Alterna la salida de sonido encendida y apagada.
Reinicia el cartucho actual. Esto equivale a recargar y hacer un inicio limpio del cartucho actual.
El botón SELECT representa el botón físico "SELECT" presente en la consola Atari 7800. La acción de este botón es específica del cartucho, pero típicamente se usa para seleccionar las configuraciones actuales del juego (como nivel de dificultad).
El botón RESET representa el botón físico "RESET" presente en la consola Atari 7800. La acción de este botón es específica del cartucho, pero típicamente se usa para terminar el juego actual (si hay un juego en progreso).
Representa el interruptor físico de dificultad izquierda presente en la consola Atari 7800. La acción de este interruptor es específica del cartucho.
Representa el interruptor físico de dificultad derecha presente en la consola Atari 7800. La acción de este interruptor es específica del cartucho.
Abre la página Tabla de Líderes Global JS7800. Esta página muestra puntuaciones altas globales y mejores jugadores. El conjunto de puntuaciones que se muestra por defecto en la tabla de líderes es el del juego que se está jugando actualmente en JS7800 (si soporta puntuaciones altas globales). Para detalles adicionales sobre la tabla de líderes global, consulta la pestaña "Puntuaciones Altas" a la izquierda. Juegos y variantes soportadas (identificadas por hashes MD5 de cartuchos crudos sin encabezado) están aquí disponibles.
Muestra este diálogo de "Ayuda".
Muestra el diálogo de "Configuración". Para detalles adicionales, consulta la pestaña de ayuda "Diálogo de Configuración" a la izquierda.
Alterna entre modo de pantalla completa y no pantalla completa.
