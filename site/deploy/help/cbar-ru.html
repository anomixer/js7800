Панель Управления

"Панель управления", которая расположена непосредственно под экраном эмулятора JS7800, используется для управления различными аспектами эмуляции, включая доступ к диалогам настроек и справки.

Кнопки и элементы управления, составляющие панель управления, описаны ниже.

Приостанавливает эмуляцию текущего картриджа. После приостановки эту кнопку можно использовать для возобновления эмуляции.
Переключает звуковой вывод включен/выключен.
Перезапускает текущий картридж. Это эквивалентно выполнению перезагрузки и чистого старта текущего картриджа.
Кнопка SELECT представляет физическую кнопку "SELECT", которая присутствует на консоли Atari 7800. Поведение этой кнопки специфично для картриджа, но обычно используется для выбора текущих настроек игры (уровни сложности и т.д.).
Кнопка RESET представляет физическую кнопку "RESET", которая присутствует на консоли Atari 7800. Поведение этой кнопки специфично для картриджа, но обычно используется для завершения текущей игры (если она в процессе).
Представляет физический левый переключатель сложности, который присутствует на консоли Atari 7800. Поведение этого переключателя специфично для картриджа.
Представляет физический правый переключатель сложности, который присутствует на консоли Atari 7800. Поведение этого переключателя специфично для картриджа.
Открывает страницу Глобальная таблица лидеров JS7800. Эта страница предоставляет возможность просматривать глобальные высокие баллы и лучших игроков. Начальный набор баллов, отображаемых в таблице лидеров, по умолчанию соответствует игре, которая в настоящее время запущена в JS7800 (если она поддерживает глобальные высокие баллы). Для получения дополнительной информации о глобальной таблице лидеров обратитесь к вкладке "Высокие баллы" слева. Необработанный дамп поддерживаемых игр и вариантов (идентифицированных по MD5-хэшу картриджа без заголовка) можно найти здесь.
Отображает этот диалог "Справка".
Отображает диалог "Настройки". См. вкладку справки "Диалог настроек" слева для получения подробной информации.
Переключает между полноэкранным и неполноэкранным режимами.
