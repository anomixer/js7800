Las diferentes formas en que los cartuchos pueden cargarse en JS7800 se detallan a lo largo del resto de esta pestaña. JS7800 debería ser compatible con la gran mayoría de ROMs de cartuchos disponibles para Atari 7800 (tanto lanzamientos originales como homebrew).

Las ROMs de cartuchos de Atari 7800 típicamente tienen una extensión de archivo .bin o .a78. La extensión .a78 es preferida ya que contiene información de encabezado que proporciona al emulador información sobre cómo debe interpretarse el cartucho particular. También vale la pena notar que JS7800 soporta ROMs de cartuchos que están contenidas dentro de archivos .zip.

Menú Desplegable de Cartuchos

El menú "Desplegable de Cartuchos" (resaltado en rojo abajo) contiene una lista de cartuchos de Atari 7800 para seleccionar.

Seleccionar Cartucho Atari 7800...

Por defecto, la lista de cartuchos se llena con una selección de excelentes títulos homebrew. Para cargar un cartucho, simplemente selecciona el elemento apropiado en el menú.

También se soportan listas de cartuchos personalizadas. Para más información sobre crear y cargar listas de cartuchos personalizadas, consulta la página de GitHub de Listas de Cartuchos.

Seleccionar Archivos Locales

El botón "Seleccionar Archivo Local" (resaltado en rojo abajo), se usa para seleccionar y cargar un archivo ROM de cartucho de Atari 7800 que está ubicado en el dispositivo local.

Seleccionar Cartucho Atari 7800...

Una vez que se presiona el botón, se mostrará un selector de archivos que permite la selección de un archivo local. Por defecto, el selector de archivos limita los archivos mostrados a extensiones .a78, .bin, .zip, y .json.

El selector de archivos local también puede usarse para seleccionar un archivo de "lista de cartuchos" (extensión de archivo .json). Seleccionar un archivo de "lista de cartuchos" actualizará el contenido del Menú Desplegable de Cartuchos. Para más información sobre listas de cartuchos, consulta la página de GitHub de Listas de Cartuchos.

Seleccionar Archivos Remotos

El botón "Seleccionar Archivos Remotos" (resaltado en rojo abajo), se usa para especificar la URL de una ROM de cartucho remota de Atari 7800.

Seleccionar Cartucho Atari 7800...

Una vez que se presiona el botón, se mostrará un diálogo de entrada que permite especificar la URL a un archivo ROM remoto. Por ejemplo, ingresar la siguiente URL causará que se cargue el juego homebrew, "bonQ":

http://raz0red.github.io/js7800/roms/bonq.a78

El selector de archivos remoto también puede usarse para seleccionar un archivo de "lista de cartuchos" remoto (extensión de archivo .json). Seleccionar un archivo de "lista de cartuchos" actualizará el contenido del Menú Desplegable de Cartuchos. Para más información sobre listas de cartuchos, consulta la página de GitHub de Listas de Cartuchos.

Arrastrar y Soltar Archivos

Arrastrar y soltar ROMs de cartuchos locales de Atari 7800 puede usarse como alternativa a seleccionar archivos vía el navegador de archivos local (ver Seleccionar Archivos Locales).

Como se demuestra arriba, los archivos ROM locales pueden cargarse simplemente arrastrándolos y soltándolos en el emulador desde el Explorador de Archivos en Windows o Finder en Mac.

Además de archivos ROM de cartuchos, los archivos de "lista de cartuchos" (extensión de archivo .json) también pueden arrastrarse y soltarse en el emulador. Arrastrar y soltar un archivo de "lista de cartuchos" causará que el contenido del Menú Desplegable de Cartuchos se actualice. Para más información sobre listas de cartuchos, consulta la página de GitHub de Listas de Cartuchos.

Arrastrar y Soltar Enlaces

Arrastrar y soltar enlaces que apuntan a ROMs de cartuchos remotos de Atari 7800 puede usarse como alternativa a especificar URLs vía el selector de archivos remoto (ver Seleccionar Archivos Remotos).

Como se demuestra arriba, las ROMs pueden cargarse simplemente arrastrando y soltando enlaces de archivos remotos en el emulador. En el ejemplo específico arriba, un enlace está siendo arrastrado desde el Foro de Programación Atari 7800 de AtariAge, una gran fuente tanto para títulos homebrew en desarrollo como completados. También vale la pena notar que algunos enlaces en el foro AtariAge requieren que el arrastre se realice vía el enlace de "conteo de descargas" versus el enlace del título del juego (como se muestra arriba).

Además de enlaces de archivos ROM de cartuchos, los enlaces de archivos de "lista de cartuchos" también pueden arrastrarse y soltarse en el emulador. Arrastrar y soltar un enlace de archivo de "lista de cartuchos" causará que el contenido del Menú Desplegable de Cartuchos se actualice. Para más información sobre listas de cartuchos, consulta la página de GitHub de Listas de Cartuchos.

Parámetros de Solicitud

JS7800 soporta el siguiente conjunto de parámetros de solicitud. Los parámetros de solicitud pueden agregarse a la URL de JS7800 para controlar varias opciones del emulador. Por ejemplo, el parámetro de solicitud cart puede usarse para forzar la carga de un cartucho específico cuando JS7800 se muestra inicialmente.

cart
El parámetro de solicitud cart puede usarse para cargar un cartucho específico cuando la página del emulador se carga inicialmente. La siguiente URL causará que se cargue el juego homebrew, "bonQ":

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78

cartlist
El parámetro de solicitud cartlist puede usarse para cargar una lista de cartuchos en el Menú Desplegable de Cartuchos cuando JS7800 se carga inicialmente. La siguiente URL causará que se llene una lista de demos homebrew en el menú desplegable de cartuchos:

https://raz0red.github.io/js7800/?cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Para más información sobre listas de cartuchos, consulta la página de GitHub de Listas de Cartuchos.

Múltiples parámetros de solicitud pueden combinarse para establecer más de una opción cuando JS7800 se carga. La siguiente URL cargará un juego específico, "bonQ", y una lista de cartuchos conteniendo demos homebrew:

https://raz0red.github.io/js7800/?cart=http://raz0red.github.io/js7800/roms/bonq.a78&cartlist=http://raz0red.github.io/js7800/roms/romlist-homebrew-demos.json

Dado que agregar parámetros de solicitud típicamente resultará en URLs largas, se recomienda el uso del servicio de acortamiento de URL de GitHub, git.io. La siguiente URL es la versión acortada del ejemplo arriba, conteniendo ambos parámetros de solicitud cart y cartlist:

https://git.io/JfcmO
