Resumen de JS7800

JS7800 es un puerto JavaScript mejorado del emulador ProSystem Atari 7800 que fue desarrollado originalmente por Greg Stanton y posteriormente portado a múltiples plataformas.

Características:

  • Seguimiento de puntuaciones altas globales (para juegos compatibles con HSC)
  • Mapeos de teclado personalizables
  • Compatibilidad con gamepads (con soporte de doble analógico para juegos estilo Robotron)
  • Soporte para pantalla completa
  • Múltiples relaciones de aspecto
  • Capacidad para habilitar/deshabilitar filtros de video
  • Soporte para arrastrar y soltar archivos locales y enlaces de archivos remotos
  • Soporte para listas de cartuchos
  • Soporte para light gun (vía mouse)
  • Soporte para módulo de expansión (XM)
  • Conmutación de bancos mejorada y detección de cartuchos

Para funcionar correctamente y sin problemas (sin lags, etc.), JS7800 requiere una versión actualizada de un navegador moderno (Chrome, Firefox, Safari) en un sistema configurado correctamente.
